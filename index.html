Words in letter blocks

If you have ever found some wooden blocks with letters on them and tried to figure out what words you can make using these cubes, look no further because this is the website to solve your problems.

Get words for your cubes:

Add letters from each cube concatenated as a word (i.e. write down the six letters without spaces ); add a comma after each cube (see example).
Do you want to get just the raw data without any formatting?

The API uses .dic files from the LibreOffice GitHub repository; I do not take responsibility for the words actually being correct or used in daily life. Some of them might not make sense (as this data source is not an actual word list but a source for a spell checker), but most of them should.
